출입 통제 기기(NFC & 릴레이) 초기 설정 가이드

IT에 익숙하지 않은 점주님들을 위한 쉽고 빠른 장치 설치 및 와이파이 연결 매뉴얼

1 장치 전원 연결 및 초기화 (리셋)

  • 3개의 장치(외부 리더기, 내부 리더기, 릴레이)를 각각 DC 5V 아답터에 연결하여 전원을 켭니다.
  • 각 기기의 뚜껑에 있는 작은 바늘 구멍에 핀(클립 등)을 넣어 내부의 Boot 버튼을 5초 이상 길게 눌러줍니다.
  • 기기가 스스로 재부팅되며 초기화가 완료됩니다.

2 스마트폰으로 기기 와이파이 연결하기

  • 스마트폰의 Wi-Fi 설정 버튼을 길게 눌러 설정 창으로 들어갑니다.
  • Wi-Fi를 껐다가 다시 켜서 목록을 새로고침 합니다.
  • 사용 가능한 Wi-Fi 목록에서 MQCafe-Setup 이라는 이름의 와이파이를 찾아 터치(선택)합니다.
  • (설정 화면이 자동으로 뜨지 않는다면) 크롬 등 인터넷 주소창에 192.168.4.1 을 입력하여 접속합니다.
  • 파란색 [Configure WiFi] 버튼을 누릅니다.
  • 화면 맨 위 목록에서 매장에서 실제로 사용 중인 공유기(Wi-Fi)를 선택하고 비밀번호를 입력합니다.

3 매장 정보 및 장치 역할 부여 (중요)

  • 비밀번호 입력 칸 바로 아래에 본사에서 지정해준 매장 ID를 정확히 입력합니다. (예: ST001)
  • 그 아래 역할(reader_action) 칸에는 기기가 설치될 위치에 맞춰 다음 중 하나를 영문 소문자로 입력합니다.

📇 entry : 밖에서 안으로 들어올 때 찍는 출입문 외부 NFC 리더기

🚪 exit : 안에서 밖으로 나갈 때 찍는 출입문 내부 NFC 리더기

⚡ relay : 자동문에 직접 연결되어 문을 열어주는 제어기(릴레이)

  • 정보를 모두 입력했다면 맨 밑의 [Save] 버튼을 눌러 저장합니다.
  • 저장이 완료되면 스마트폰의 [이전] 화면으로 돌아가, 스마트폰의 와이파이를 다시 원래 매장 와이파이로 연결해 줍니다.
⚠️ 주의: 위 [2번 ~ 3번] 과정을 3개의 기기(entry, exit, relay) 모두 동일하게 반복하여 설정해 주셔야 완벽하게 세팅이 끝납니다!

4 실제 출입문(NFC 카드) 연동 테스트

  • 스마트폰 카메라로 매장 앞에 비치된 고객용 QR 코드를 스캔하여 웹사이트에 접속합니다.
  • 고객 입장이 되어 이용권(시간권 등)을 1개 구매하고 결제를 완료합니다.
  • 결제 완료 후 나오는 이용 화면에서 [출입용 카드 간편 연동하기] 버튼을 누른 뒤 [연동 시작]을 누릅니다.
  • 60초 이내에 출입증으로 사용할 임의의 플라스틱 카드(교통카드 등)를 외부 출입문 리더기(entry)에 태그합니다.
  • 스마트폰에 '등록 완료' 화면이 뜨면 성공입니다!
  • 이제 방금 그 카드를 다시 NFC 리더기에 대면, 자동문(릴레이)이 찰칵 하고 열리는지 확인하시면 모든 설치가 끝납니다.
